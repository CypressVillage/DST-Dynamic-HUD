# [DST] Dynamic HUD

本模组允许你根据当前环境动态切换不同的HUD。

github: [CypressVillage/DST-Dynamic-HUD](https://github.com/CypressVillage/DST-Dynamic-HUD)

## 功能

- 根据人物所处环境切换不同 HUD
- 按下快捷键切换 HUD

## 注意事项

> [! warning]
> 必须同时启用对应的HUD模组，切换HUD功能才会生效
